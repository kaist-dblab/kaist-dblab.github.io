Goal　 
  • A General Purpose Multi-User Main Memory Object Storage System
　　 

Features and Characteristics　 

  • All the Objects reside in Main Memory
  • Supporting Variable Length Objects
  • T-tree Indexing
  • Concurrent Access for Multiple Users
  • Robust WAL-Based Crash Recovery with Record (Object) Level Locking
  • Multi-threaded Architecture

 

Application Areas　 

  • Real-Time Database Management Systems
  • Home Location Registration(HLR)
  • Mobile Computing
  • Factory Automation(FA)
  • CIM
    　　　 　　　　　　 
    　

Expected Results 

  • Core technological know-how for Real-Time Database Management System
